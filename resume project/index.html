  • Android Auto Certification
    • Android Auto (AA) Certification is the process which ensures that the product integrated in the Head Unit is ready for production. This includes Projection Compatibility Test Suite (PCTS), Qsuite (For speech test), Voice recognition Response Time(VRRT), Performance Test (Knob and Touch), Plugbot (Automated Lab Testing), Manual USB Reliability Test, Sensor Test, Drive Integration Test, Native Navigation Test and AOAP.
  • Carplay
    • CarPlay is a technology promoted by Apple to connect the automotive Head Unit displays to the Apple Smartphone. CarPlay allows the Apple smartphone to project a screen to the Head Unit display over USB. The Head Unit can launch and control the applications exposed by the smartphone using touch and knobs / keys. Apple speech assistant software 'Siri' can also be invoked from the Head Unit steering wheel control. CarPlay also allows navigation application, along with voice prompts for directions, to be launched and controlled from the Head Unit. CarPlay is part of iOS versions 7.1 and above.
